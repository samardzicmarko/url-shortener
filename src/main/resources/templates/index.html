Installation and guide

java -ar

If necessary install dependencies from pom.xml with "mvn install"

Application will be hosted on http://localhost:8080

Open your favorite API REST Client (Insomnia, Postmen..) and create account to test the application

API Specification

Opening Account

RequestType: POST
Uri: /account
Required Field: "accountId"
Example: { "accountId": "Marko" }
Response: { "Password:": "bRa9UpXm", "Succes:": "true", "Description:": "Your account is opened" }

Register URL

RequestType: POST
Uri: /register
Authorization: Basic Authorization. accountId, password
Required Field: "uri"
Response: "Shortener url" (example: http://localhost:8080/pyxftL)
Example: {"uri": "https://stackoverflow.com/questions/48575559/displaying-index-html-page-using-controller-spring-boot"}

View Url

RequestType: GET
Uri /{shortUrl}
Response Redirected url
Example: http://localhost:8080/pyxftL/pyxftL redirects to original url

Statistic

RequestType: GET
Uri /statistic/{accountId}
Response: original url, view Integer
Example: {"https://stackoverflow.com/questions/48575559/displaying-index-html-page-using-controller-spring-boot": 3 "https://spring.io/guides/gs/rest-service/": 53}
